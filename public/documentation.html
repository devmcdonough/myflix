myFlix Documentation

Here are some instructions on how to use the Movie API.

Request URL HTTP Method Request body data method Response body data method
Return all movies to user /movies GET None An array of JSON objects holding data about all the movies
Return data about a single movie by title to user /movies/[title] GET none JSON object holding data about movie requested by title
Return data about a genre by name/description movies/genre/[genre] GET none An array of JSON objects holding data about all movies in the requested genre
Filter movies from a director by name /movies/directors/[director name] GET none JSON object holding data about a director
Allow new users to register /users POST JSON object containing user details
Example:
{ id: 1,
username: 'Sam Samon',
email: 'samon@sam.org',
password: Hello123
} 		JSON object with details of outcome, including user ID.
JSON object containing user details
Example:

Allow users to update their user info (username) users/[userID]/username PUT none A message indicating the user info was updated
Allow users to add a movie to their list of favorites user/[userID]/movies/[movieID] PUT none A message indicating whether the movie was successfully added
Allow users to remove a movie from their list of favorites /user/[userID]/movies/[movieID DELETE none A message indicating whether the movie was successfully removed
Allow existing users to deregister /user/[userID] DELETE none A message indicating whether the user successfully deregistered
